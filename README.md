# IT Helpdesk Portfolio

This repository contains multiple IT Helpdesk projects demonstrating IT support and administrative skills.

## Projects
1. **Helpdesk Ticket System** – Tracks support tickets, priorities, and resolutions.
2. **Helpdesk Knowledge Base** – Troubleshooting documentation for common IT issues.
3. **User Account Management Simulation** – Simulates user account creation, admin actions, and access control.

Each project folder contains a README with more details.
