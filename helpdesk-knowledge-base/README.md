# Helpdesk Knowledge Base



This repository contains **troubleshooting documentation** for common IT support issues.



## Contents

- **login-password-reset.md** – Resolving login and password problems.

- **email-issues.md** – Fixing email client and syncing issues.

- **network-issues.md** – Troubleshooting Wi-Fi, VPN, and network problems.

- **software-issues.md** – Handling application crashes and installation failures.

- **hardware-issues.md** – Fixing keyboard, mouse, and peripheral problems.



## Purpose

This knowledge base is designed to **help IT support staff quickly resolve issues** and serve as a reference for new employees.
