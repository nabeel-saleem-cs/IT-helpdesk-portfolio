\# Network Issues



\## Common Problems

\- Wi-Fi not connecting

\- VPN not working

\- Slow internet



\## Troubleshooting Steps

1\. Confirm Wi-Fi enabled.

2\. Restart network adapter.

3\. Forget \& reconnect to network.

4\. Restart router if needed.

5\. Check IP configuration.



\## VPN Problems

1\. Verify VPN credentials.

2\. Restart VPN client.

3\. Check firewall settings.

4\. Reinstall VPN software if necessary.



\## Outcome

Network connection restored.
