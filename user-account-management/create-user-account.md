\# Creating a New User Account



\## Purpose

This procedure outlines how IT support staff create new user accounts.



\## Steps



1\. Receive account creation request from HR or manager.

2\. Verify employee information.

3\. Create account in directory system.

4\. Assign temporary password.

5\. Assign department and role.

6\. Enable MFA.

7\. Send login credentials securely.



\## Outcome

User account successfully created and ready for first login.
