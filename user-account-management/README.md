# User Account Management Simulation

This project simulates common IT support tasks related to **user account administration**.  
It demonstrates how helpdesk or IT support staff manage user accounts, permissions, and account access issues.



## Repository Structure

user-account-management-simulation

data/
- user_accounts.xlsx – Contains user account dataset and summary report.

procedures/
- create-user-account.md – Steps for creating new user accounts.
- reset-password.md – Password reset procedure.
- unlock-account.md – Process for unlocking user accounts.
- assign-permissions.md – Guidelines for assigning user permissions.

README.md – Project overview.



## Data File

The Excel file includes two sheets:

Data  
Contains a sample dataset of user accounts including:
- User ID
- Name
- Department
- Role
- Account Status
- Last Login
- MFA Enabled

Summary  
Contains metrics and analysis derived from the dataset, such as:
- Total user accounts
- Active vs locked vs disabled accounts
- MFA adoption statistics
- Charts visualizing account status distribution



## Purpose

This project demonstrates key responsibilities of IT support teams including:

- User account creation
- Password reset support
- Account unlocking
- Access permission management
- Basic reporting on account status

These tasks are commonly performed in enterprise identity management systems such as Active Directory.
