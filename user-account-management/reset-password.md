\# Resetting a User Password



\## Purpose

Used when a user forgets their password or cannot log in.



\## Steps



1\. Verify user identity.

2\. Locate user account in admin console.

3\. Reset password to temporary password.

4\. Require password change at next login.

5\. Confirm user can access account.



\## Outcome

User regains access to their account.
